1、使用npm init后，生成package.json文件

2、新建webpack.config.js指定入口、出口

3、然后直接执行webpack即可生成bundle.js

4、使用npm run xxx可以执行package.json文件中的script中的命令，例如npm run build

5、局部打包，局部安装一个webpack，因为有肯能需要和全局的webpack不同的版本，使用命令：npm install webpack@3.6.0 --save-dev

--save-dev为了开发时依赖，运行时不依赖

安装完成后，发现package.json中多一个devDependencies属性，意思是开发时依赖

然后执行node_modules/.bin/webpack就是局部的webpack打包了（配置在package.json中script命令：webpack，会优先使用局部的，局部没有再使用全局的）

6、-D相当于--save-dev的简写，-S相当于--save的简写
